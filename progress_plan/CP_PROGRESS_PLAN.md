- Pick one new technique from the "Techniques Queue" below
- Read 1 tutorial + 1 editorial explanation
- Understand the pattern, don't code yet

#### **Tuesday: PRACTICE**
- Find 2-3 problems using Monday's technique
- Start implementing, struggle is expected!

#### **Wednesday: SOLVE**
- Complete at least 1 problem with the technique
- If stuck >2 hours, read editorial but implement yourself

#### **Thursday: OPTIMIZE**
- Review others' solutions for the same problems
- Try to improve your solution (time/memory/elegance)

#### **Friday: COMPETE**
- Join a live contest or do a virtual one
- Look for opportunities to use the week's technique

#### **Weekend: EXPLORE**
- Free choice! Solve anything interesting
- Or work on your visualization projects
